Pneumoty

Jully 3, 2021

Pneumoty is my Bachelors's degree graduation project which is a social network dedicated for doctors (pulmonists to precise) where we collaborated with doctors at CHU Oran Hostpital to deliver to them this website which gonna help them communicate with each other and take quick action again serious and urgent cases

It is built using the MERN stack (MongoDB, Express, React, Nodejs). I used Redux as a state management for React and Tailwindcss as a css framework an SocketIO for the chat application. I also used cloudinary to upload images
